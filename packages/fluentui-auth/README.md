# @raftlabs/fluentui-auth

Generate auth based pages and components in your nextjs application

To Run

```
nx generate @raftlabs/fluentui-auth:generate-files --project=project-name
```

The above command will create the following files in `src` folder

```
 project-name/src/components/AuthLayout.tsx
 project-name/src/components/ChangePassword/ChangePassword.tsx
 project-name/src/components/ChangePassword/change-password.schema.ts
 project-name/src/components/ForgotPassword/ForgotPassword.tsx
 project-name/src/components/ForgotPassword/forgot-password.schema.ts
 project-name/src/components/ResetPassword/ResetPassword.tsx
 project-name/src/components/ResetPassword/reset-password.schema.ts
 project-name/src/components/SignIn/SignIn.tsx
 project-name/src/components/SignIn/sign-in.schema.ts
 project-name/src/components/SignUp/SignUp.tsx
 project-name/src/components/SignUp/sign-up.schema.ts
 project-name/src/pages/activate-account.tsx
 project-name/src/pages/change-password.tsx
 project-name/src/pages/forgot-password.tsx
 project-name/src/pages/index.tsx
 project-name/src/pages/reset-password.tsx
 project-name/src/pages/sign-in.tsx
 project-name/src/pages/sign-up.tsx
 project-name/src/pages/styles.css
 project-name/src/pages/url-request-handler.tsx
```
